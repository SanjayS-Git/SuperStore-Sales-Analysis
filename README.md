# SuperStore-Sales-Analysis
A complete Business and Data Analysis project performed on the Superstore dataset using Microsoft Excel. This project focuses on analyzing sales performance, profit trends, customer segments, regional growth, product categories, discount impact, and shipping efficiency to generate meaningful business insights and data-driven recommendations. The analysis includes data cleaning, Pivot Tables, Pivot Charts, KPI tracking, dashboard creation, profit margin evaluation, identification of loss-making products, and visualization of key performance metrics to support strategic business decision-making.
